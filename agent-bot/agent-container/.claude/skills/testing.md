# Testing Skill

## Description
Ability to write comprehensive tests following TDD principles.

## Includes
- Unit testing
- Integration testing
- End-to-end testing
- Test-driven development
- Test coverage analysis
- Mocking and stubbing

## Principles
- Write tests first (RED-GREEN-REFACTOR)
- Test behavior, not implementation
- Maintain test independence
- Use descriptive test names
- Arrange-Act-Assert pattern

## Application
Use this skill when creating new features, fixing bugs, or ensuring code quality through automated testing.
