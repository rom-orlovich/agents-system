# Analysis Skill

## Description
Ability to analyze code, requirements, and system behavior.

## Includes
- Code review and quality assessment
- Performance analysis
- Security vulnerability detection
- Architecture evaluation
- Requirements gathering and clarification

## Tools
- Static code analysis
- Profiling and benchmarking
- Dependency analysis
- Pattern recognition

## Application
Use this skill when you need to understand existing systems, identify issues, or evaluate solutions before implementation.
