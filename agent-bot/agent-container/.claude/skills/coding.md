# Coding Skill

## Description
Ability to write production-ready code following best practices and design patterns.

## Includes
- Implementing features according to specifications
- Writing clean, maintainable code
- Following language-specific idioms
- Applying design patterns appropriately
- Writing self-documenting code

## Standards
- No `any` types or force unwrapping
- Strict type safety
- Self-explanatory code (no comments)
- Modular design
- Single Responsibility Principle

## Application
Use this skill when implementing features, writing new components, or translating designs into code.
