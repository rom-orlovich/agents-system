# Implement Command

## Syntax
`@agent implement <feature>`

## Description
Implement a feature or fix following TDD principles.

## Parameters
- `feature`: Description of what to implement

## Behavior
1. Understand requirements
2. Write tests first (TDD)
3. Implement feature
4. Ensure all tests pass
5. Refactor for quality
6. Create commit with changes

## Examples
- `@agent implement user authentication`
- `@agent implement dashboard analytics`
- `@agent implement retry logic`

## Agent Assignment
Routes to: **coding** agent with **coding** and **testing** skills

## Requirements
- Must follow TDD (tests first)
- Must pass all existing tests
- Must follow type safety rules
- Must be self-explanatory code
